# Task 1 – JSON Task CLI

A prototype command-line Task Manager that lets you **store**, **list**, and **search** tasks in a JSON data file.

---

## 📁 Project Overview
This is a minimal, portable Python CLI application developed as part of the CSC299 project series.  
It saves your tasks into a local `tasks.json` file, which allows you to keep track of to-do items without any database setup.

---

## ⚙️ Requirements
- Python 3.8 or newer  
- Works on Windows, macOS, and Linux  
- No external libraries required (only Python built-ins)

---

## 🚀 How to Run
From inside the `tasks1` directory, run the following commands in **PowerShell** or any terminal.

```bash
python task_cli.py add "Read PKMS paper"
