# Final Project — PKMS + Task Manager + Chat Interface + AI Agent

Run the program:

    uv run main.py

This CLI lets you:

-Add tasks
-List tasks
-Search tasks
-Sort tasks
-Mark tasks done
-Edit tasks
-Delete tasks
-Clear completed tasks
-Add notes
-List notes
-Search notes
-Delete notes
-Link notes to tasks
-View tasks with their linked notes
-Use an AI note-summary tool
-Track task deadlines
-Track high/medium/low priorities
-Organize tasks with tags
-View analytics using stats
-See overdue tasks
-See tasks due today
-See tasks due this week
-Monitor note–task relationships
-Use a chat-style interface for all actions
-Save everything automatically to JSON
-Reload tasks and notes on program start
